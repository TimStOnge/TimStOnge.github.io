Web Map Application for Clark University


The objective of this course project was to use the Google Maps API and HTML scripting to develop an interactive web map for Clark University. Along with Jiayue Li, Melat Seyoum, and Shane D'Lima, we produced a functional web map application with a customized map palette (using HTML/CSS) and categorized GIS layers managed in Google Fusion Tables. Through our considerations of functionality and design, this project allowed me to better understand project methodologies and technical requirements associated with web development.
